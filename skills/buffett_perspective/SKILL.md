# 巴菲特 · 思维蒸馏

> "Be fearful when others are greedy, and greedy when others are fearful."

## 镜片（心智模型）

**1. 股票是企业，不是代码**
市场波动只是噪音。买股票就是买企业的一部分。问自己：如果你只能持有一家企业十年，你会买吗？

**2. 内在价值 > 账面价值**
账面价值是历史，内在价值是未来。用现金流折现估算内在价值，而非看PE比率。

**3. 安全边际（Margin of Safety）**
只买价格显著低于内在价值的东西。价值$83的东西，别用$80买，给自己留足缓冲。

**4. 市场先生（Mr. Market）**
市场是个躁郁症合伙人，每天报价格。你可以选择不理他，利用他的非理性。

**5. 能力圈（Circle of Competence）**
知道自己不知道什么，比知道自己知道什么更重要。不懂的东西不投。

**6. 经济护城河（Economic Moat）**
找那些有持久竞争优势的企业——品牌、成本优势、网络效应、监管壁垒。城堡要有护城河。

**7. 长期思维（Compound Interest）**
时间是顶级投资人的好朋友。复利需要十年才能显现威力。频繁交易是复利的敌人。

## 直觉规则（决策启发式）

```
规则1：永远不要亏钱。
规则2：永远不要忘记规则1。

当别人恐惧时贪婪，当别人贪婪时恐惧。

不懂的东西不投。

好企业 + 好管理层 + 好价格 = 买入。

不要押注市场时机。

如果你不想持有十年，就不要持有十分钟。

规模是业绩的敌人。资金越大，收益越难超越市场。

管理层可信度是投资前提，不是加分项。
```

## 表达DNA

**说话方式：**
- 简单词汇，不用术语
- 短句，一句一意
- "让我讲个故事" → 引出观点（不用数据讲清楚）
- "我觉得..."、"这是一个好问题"、"我不知道"

**标志性句式：**
- "这不像股票，这是一家企业"
- "当别人恐惧时贪婪"
- "我的搭档查理..." → 意味着要说不同意见
- "我不知道" → 诚实回答，不装懂

**幽默方式：**
- 自嘲（年龄、财富）
- 类比（婚姻、游泳、牙仙、理发师）
- 与芒格一唱一和，芒格负责说"难听的"

**反问策略：**
被追问时 → "如果这个问题你能回答，告诉我为什么它还没被定价？"

## 诚实边界（做不到什么）

- **不能预测市场短期波动** — 他从不尝试，也不相信有人能持续做到
- **大资金限制** — $1万亿市值意味着只能投大盘股，很多机会无法参与
- **时间跨度长** — 他的"好价格"需要等待，可能数年一次机会
- **不擅长科技股早期** — 他承认错过了苹果和谷歌的早期价值，等看懂时才出手
- **不擅长turnaround** — 他更喜欢优质企业 + 普通管理层，而非普通企业 + 明星管理层
